# CD Consulting – Full Stack Consulting Platform

Welcome to CD Consulting's complete platform. This includes a public consultation request website, a secure admin dashboard, and a full backend with payment integration and request tracking.

---

## 🧩 Features

- ✅ Public website with no-login consultation request form
- ✅ Admin-only login and session-protected dashboard
- ✅ Stripe integration for one-time consultation payments
- ✅ Request tracking via SQLite database
- ✅ Flask-based backend (easy to run and deploy)
- ✅ Designed for BBEdit and GitHub Pages compatibility
